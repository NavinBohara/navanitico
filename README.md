# 🚀 NavaNiti — AI-Powered Financial Literacy Platform

NavaNiti is an AI-powered financial literacy and budget planning platform designed to help users improve financial awareness, manage expenses, and make smarter financial decisions through interactive tools, AI assistance, live financial news, and scam awareness features.

The platform combines financial education, budgeting, calculators, AI chatbot support, gamified learning, and real-time finance updates into a single user-friendly system.

# ⚙️ Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone (https://github.com/NavinBohara/navanitico)
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd NavaNiti
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Configure Environment Variables

Create a `.env` file in the root directory:

```env
VITE_SUPABASE_URL=*PVT*
VITE_SUPABASE_ANON_KEY=*PVT*
VITE_GEMINI_API_KEY=*PVT*
VITE_FINNHUB_API_KEY=*PVT*
```

---

## 5️⃣ Start Development Server

```bash
npm run dev
```
